      server 127.0.0.1;    //delete the port

webpack-dev-server error

  • check the upstream config and the proxy_pass config
  • check the /etc/hosts file
      sudo subl /etc/hosts
  • run gulp 
      gulp [hot] [dev]
  • run npm start
      npm start
  • check the server port
       sudo netstat -ap | grep 12010
